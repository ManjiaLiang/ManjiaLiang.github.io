Quick Start

环境配置

Node.js和Git是安装Hexo的前提；npm是JavaScript运行时Node.js的默认程序包管理器。

建站

在桌面创建一个空文件夹Blog，用于存放博客框架，并在Blog路径下初始化 Hexo。

部署至Github

  1. 创建Github个人token

自2021年8月13日起，Git不再支持密码方式验证，而是建议使用token，因此首先需要创建个人token，流程如下：

  • github.com >>> 头像 >>> Settings >>> Developer settings >>> Personal access tokens >>> Generate new token
  • Note : Blog/Expiration : no expiration/Select scopes : repo, admin:repo_hook, delete_repo >>> Generate token
  • 注：务必第一时间复制并记录下token，因为只有首次生成token时才能看到token信息。
  1. token的使用
  • 在通过HTTPS执行Git操作时，输入token，而不是密码
  • 将钥匙串中更新凭据，用token替换旧密码
