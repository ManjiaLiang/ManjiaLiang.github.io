一个博客的诞生

环境配置

安装node

brew install node

安装git

brew install git

安装npm

brew install npm

Review

node -v
git --version
npm -v

安装hexo

sudo npm install -g hexo-cli

用Hexo搭建博客

在桌面创建一个空文件夹blog

在blog路径下初始化一个博客

cd Desktop/blog
hexo init

启动hexo
hexo s

根据提示在浏览器中输入http://localhost:4000

至此，博客网页可以在本地运行与查看

命令行下ctrl+c，退出命令，仍然在blog目录下

部署至Github

登录Github，新建一个仓库New repository，仓库名称必须为：昵称.github.io

在blog路径下安装git部署插件：npm install hexo-deployer-git --save

更改blog文件夹下，_config.yml的配置

回到命令行部署到远端

hexo d
