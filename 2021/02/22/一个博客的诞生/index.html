一个博客的诞生

发表于 2021-02-22 更新于 2021-02-23

环境配置

安装node

brew install node

安装git

brew install git

安装npm

brew install npm

安装hexo

sudo npm install -g hexo-cli

测试安装情况

用Hexo搭建博客

在桌面创建一个空文件夹blog

文件夹是用来存放博客框架的。

在blog路径下初始化Hexo

启动hexo

hexo s

根据提示，在浏览器中输入http://localhost:4000
至此，博客网页可以在本地运行，可以查看到默认生成了一篇博文
查看完毕，可以回到终端，按下ctrl+c，退出命令，返回blog目录

部署到Github

想要博客网站可以通过互联网访问，需要我们将它部署到远端服务器上。Github Page管理方便、免费、具有无限流量，所以我们选它。

在Github上新建一个仓库

登录Github，新建一个仓库New repository，仓库名称必须为：昵称.github.io。

在blog路径下安装deployer插件

npm install hexo-deployer-git --save

更改_config.yml的配置

打开blog路径下的_config.yml，做以下修改，repo的值为Github上repo的地址。

部署到远端

hexo d
